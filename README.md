# House Price Prediction

## Overview
Built and compared 3 ML models to predict California house prices.

## Tools Used
Python, Scikit-learn, Pandas, Matplotlib, Seaborn

## Results
| Model | R² Score |
|-------|----------|
| Linear Regression | ~0.60 |
| Decision Tree | ~0.72 |
| Random Forest | 0.8051 ✅ Best |

## Dataset
California Housing Dataset — Scikit-learn built-in
